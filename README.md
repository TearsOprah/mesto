# Проект: Место

### Интерактивной страница, куда можно добавлять фотографии, удалять их и ставить лайки на свои и чужие карточки


**В работе оттачивались навыки:**

* Верстка с использованием flex-box и grid layout
* Адаптивная верстка
* БЭМ
* Работа с DOM (манипуляции с классами CSS, реакция на действия пользователя)
* ООП
* Валидация форм
* Адаптивность
* Сборка проекта с Webpack
* Использование асинхронных функций
* Работа с api

`npm install` - установка зависимостей

`npm run dev` - запуск сервера

`npm run build` - запуск сборки проекта
